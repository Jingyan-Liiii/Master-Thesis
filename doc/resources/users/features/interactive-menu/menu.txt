  alldecompositions     write all known decompositions to files (format is given by file extension, e.g. {dec,blk,ref,gp,tex})

                   -->  write all known original decompositions to files (format is given by file extension, e.g. {dec,blk,ref,gp,tex})
  allpresolveddecompositions
                   -->  write all known presolved decompositions to files (format is given by file extension, e.g. {dec,blk,ref,gp,tex})

  familytree            write all (partial) decompositions contained in family tree to files (.gp/.tex) and create family tree file (.tex)

  history               write command line history to a file (only works if SCIP was compiled with 'readline')

  mip                   write current node MIP relaxation in LP format to file
  mipstart              write mip start to file
  nlp                   write current node NLP relaxation to file

  reportdecompositions  write report of all finished decompositions to LaTeX format
  selecteddecompositions
                   -->  write selected (in "explore" submenu) decompositions to files (format is given by file extension, e.g. {dec,blk,ref,gp,tex})

  transmatrix           write gnuplot file showing the nonzero structure of the transformed (presolved) matrix
